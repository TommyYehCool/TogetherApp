## 🎯 快速決策樹

### 我應該用哪種方式測試？

```
開始
  │
  ▼
有實體 Android/iOS 手機？
  │
  ├─ 是 → 使用實體手機 ⭐ (最佳選擇)
  │
  └─ 否
      │
      ▼
  已安裝 Android Studio？
      │
      ├─ 是 → 使用 Android 模擬器 ✅ (推薦)
      │
      └─ 否
          │
          ▼
      已安裝 Chrome？
          │
          ├─ 是 → 使用 Chrome 瀏覽器 ⚡ (最快)
          │
          └─ 否 → 請先安裝 Chrome 或 Android Studio
```

## 🚀 執行流程

### 方式 A：Chrome 瀏覽器

```
flutter config --enable-web
    │
    ▼
flutter run -d chrome
    │
    ▼
Chrome 啟動
    │
    ▼
應用程式載入
    │
    ▼
查看基本 UI
    │
    ▼
測試互動功能
```

### 方式 B：Android 模擬器

```
開啟 Android Studio
    │
    ▼
啟動 AVD Manager
    │
    ▼
建立/啟動模擬器
    │
    ▼
flutter devices (確認裝置)
    │
    ▼
flutter run
    │
    ▼
應用程式安裝到模擬器
    │
    ▼
測試完整功能
```

### 方式 C：實體手機

```
開啟手機開發者選項
    │
    ▼
啟用 USB 偵錯
    │
    ▼
USB 連接電腦
    │
    ▼
flutter devices (確認裝置)
    │
    ▼
flutter run
    │
    ▼
應用程式安裝到手機
    │
    ▼
測試真實體驗
```
